# Default Difficulty
#-------------------------------------------
To revert to the original difficulty you will need to:

1.
 - Copy ClearCountCorrectParam.csv to Yapped\Projects\ExampleMod\CSV\ER or Yapped\res\GR\Data for older versions.

2. 
 - Open regulation.bin with Yapped.

3.
 - On the left, select ClearCountCorrectParam.

4.
 - On the top menu bar, select Field Data->Import Data.

5. 
 - Ensure NG+0 has all 0 values.

6.
 - Save.
